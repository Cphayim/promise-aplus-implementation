# promise-aplus-implementation

## 说明

目录包含了一个符合 [Promises/A+](https://promisesaplus.com/) 规范的实现以及一个简化版实现

```
.
├── promise-easy.js
└── promise.js
```

## 如何测试？

`package.json` 依赖中包含了 Promise/A+ 的测试套件 [promises-tests](https://github.com/promises-aplus/promises-tests)


```sh
$ npm install
$ npm test
```
